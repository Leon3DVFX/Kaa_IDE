Main Program

General information

Kaa_IDE is designed for writing and testing Python code within Digital Content Creation (DCC) programs.

The program works stably in 3 DCC: Autodesk Maya, Autodesk 3dsMax, SideFX Houdini.

Currently, the program is in the development and early access stage. Some features may not work as expected, and bugs may be present.

General scheme

Main components

When you launch the application, the program icon shown in the image below appears.

Main button.
  1. Clicking on the icon collapses/expands the main program window.
    This icon allows you to drag the entire external structure.
  2. Close the program.

    Note: Closing the program using this button is correct and will not result in the loss of internal tabs or text. The program automatically saves files when closing.

  3. Call the file saving/loading window.
Saving/loading window.

All components of this widget lead to the calling of dialogs (save - load).

  1. Save the text of the current tab with the .py extension.
  2. Save the text of all tabs to a file with the .kaa extension.

    Note: The .kaa format is a custom format for this program that allows you to save all open tabs in one file.

  3. Loads text files (.txt, .py) and creates new tabs based on their names.
  4. Loads a .kaa file and restores all tabs saved in it. Overwrites the current state.
Main program window.
  1. Main tabs (multi-window mode). Right-clicking a tab displays a context menu that allows you to create a new tab or rename the current one. On the right side of the tab is a mini-button to close the tab.

    Note: The program supports multi-window mode. You can create multiple tabs and switch between them. It is not possible to delete the last (only) tab. Tab names are used when saving in .py format.

  2. The main window for writing Python code. It allows for full code editing, with all the conveniences discussed below.
  3. Additional buttons.

    • env a button under development that logs objects loaded into the internal environment. A separate object editor is planned.
    • Show\Hide Block lines
    • Unfold all

  4. Line numbering area. Allows you to set navigation bookmarks and control block folding.
  5. Information line. Contains information about the interpreter version and the current cursor position. Allows you to quickly move the cursor.
  6. Log output window. This window allows you to monitor objects. Exceptions are also displayed in this window.
Line numbering area.
  1. Left-clicking the numbers displays navigation tabs. These tabs facilitate hotkey navigation and also visually highlight important sections of code.

    Note: Alternating left-clicks changes the colors until reset. Alt + left-click to step back. Ctrl + left-click to clear the tab.

  2. The "+" and "-" icons allow you to collapse code blocks (hide them). Icons appear when you hover over them.

    Note: The text doesn't disappear, it's just hidden. "-" collapses the block, "+" expands it. If a code block is collapsed, a "folded" icon appears.
Hotkeys

Note: Hotkeys are only available in the main program window. When the main program window is not active, hotkeys will not work.

General hotkeys

Ctrl + Enter
Ctrl + Return

Runs Python code written in the main window (editor).

The objects created during the process are placed in an isolated environment (sandbox).

Depends on the selection:

- If a selection is made, only the selected code will be executed (considering the environment).
- If no selection is made, the entire code in the editor will be executed.

The location of the cursor does not matter.

Ctrl + E

Environmental check.

- If objects were created (overwritten) as a result of running the code, they will be placed in the environment. A list of created objects is output to the log.
- Otherwise, the log will indicate that the environment is clean.

Ctrl + R

Environment clear.

Completely clears the environment of previously created objects.

F1

Help with the current DCC API.

Opens the home web browser and displays the main page with the current DCC API.

Note: The editor is a hybrid between the IDE and REPL models and can be used for detailed analysis of created objects.

Text editing hotkeys

Tab

Equivalent to 4 * spaces.

Supports multi-line mode.

Shift + Tab

Reverse action.

Estimates the number of spaces before text. Removes indentation in 4-space increments.

Supports multi-line mode.

Ctrl + D

Duplicate line.

Clicking duplicates the current line and places it below the cursor, moves the cursor to the new line.

Supports multi-line mode.

Ctrl + /

Comment out the line (remove comment).

Creates a comment (#) from the line where the cursor is located. Pressing it again removes the comment.

Supports multi-line mode.

Ctrl + +
Ctrl + -

Change font sizes.

(Ctrl + +) increases the font size.
(Ctrl + -) decreases the font size.
