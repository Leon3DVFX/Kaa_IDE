Main Program

Kaa_IDE is designed for writing and testing Python code within Digital Content Creation (DCC) programs.

The program works stably in 3 DCC: Autodesk Maya, Autodesk 3dsMax, SideFX Houdini.

Currently, the program is in the development and early access stage. Some features may not work as expected, and bugs may be present.
