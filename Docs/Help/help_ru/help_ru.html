Основная программа

Общая информация

Kaa_IDE предназначен для написания и тестирования кода Python в программах цифрового создания контента (DCC).

Программа стабильно работает в 3 DCC: Autodesk Maya, Autodesk 3dsMax, SideFX Houdini.

В настоящее время программа находится на стадии разработки и раннего доступа. Некоторые функции могут работать не так, как ожидалось, и могут присутствовать ошибки.

Общая схема

Основные компоненты

При запуске приложения появляется иконка, показанная на изображении ниже.

Главная кнопка.
  1. Нажатие на значок сворачивает/разворачивает главное окно программы.
    Этот значок позволяет перетаскивать всю внешнюю структуру целиком.
  2. Закрыть программу.

    Примечание: Закрытие программы с помощью этой кнопки корректно и не приведет к потере внутренних вкладок или текста. Программа автоматически сохраняет файлы при закрытии.

  3. Вызвать окно сохранения/загрузки файла.
Окно сохранения/загрузки.

Все компоненты этого виджета ведут к вызову диалогов (сохранение - загрузка).

  1. Сохранить текст текущей вкладки с расширением .py.
  2. Сохранить текст всех вкладок в файл с расширением .kaa.

    Примечание: Формат .kaa является пользовательским форматом для этой программы, который позволяет сохранить все открытые вкладки в одном файле.

  3. Загружает текстовые файлы (.txt, .py) и создает новые вкладки на основе их имен.
  4. Загружает файл .kaa и восстанавливает все вкладки, сохраненные в нем. Перезаписывает текущее состояние.
Окно основной программы.
  1. Основные вкладки (многократный режим окон). Правый клик по вкладке показывает контекстное меню, позволяющее создать новую вкладку или переименовать текущую. Справа от вкладки находится мини-кнопка для закрытия вкладки.

    Примечание: Программа поддерживает многократный режим окон. Можно создавать несколько вкладок и переключаться между ними. Невозможно удалить последнюю (единственную) вкладку. Имена вкладок используются при сохранении в формате .py.

  2. Окно основной программы для написания кода на Python. Оно позволяет выполнять полное редактирование кода, с учетом всех удобств, обсуждаемых ниже.
  3. Дополнительные кнопки.

    • env кнопка для регистрации и работы с объектами, помещенными во внутреннее окружение. Находится в разработке.
    • Показать/Скрыть блоки.
    • Развернуть все.

  4. Область нумерации строк. Позволяет устанавливать закладки навигации и управлять сворачиванием блоков.
  5. Информационная строка. Содержит информацию о версии интерпретатора и текущей позиции курсора. Позволяет быстро перемещать курсор.
  6. Окно вывода логов. Это окно позволяет отслеживать объекты. Исключения также отображаются в этом окне.
Область нумерации строк.
  1. ЛКМ по номеру строки устанавливает/удаляет закладку навигации. Данные элементы упрощают навигацию по коду с помощью горячих клавиш, а также визуально выделяют важные разделы кода.

    Примечание: Поочередное нажатие ЛКМ меняет цвета закладок до сброса. Alt + ЛКМ для шага назад. Ctrl + ЛКМ для очистки закладок.

  2. Иконки "+" и "-" позволяют сворачивать блоки кода (скрывать их). Иконки появляются при наведении курсора мыши.

    Примечание: Текст не исчезает, он просто скрывается. "-" сворачивает блок, "+" раскрывает его. Если блок свернут, появляется иконка-маркер "folded".
Горячие клавиши

Примечание: Горячие клавиши доступны только в главном окне программы. Когда главное окно программы не активно, горячие клавиши не будут работать.

Основные горячие клавиши

Ctrl + Enter
Ctrl + Return

Запускает Python-код, написанный в главном окне (редакторе).

Объекты, созданные в процессе, помещаются во внутреннее окружение (песочница).

Операция зависима от выделения:

- Если выделен фрагмент кода, будет выполнен только этот фрагмент (с учетом окружения).
- Если выделения нет, будет выполнен весь код в редакторе.

Расположение курсора не имеет значения.

Ctrl + E

Проверка окружения.

- Если объекты были созданы (перезаписаны) в результате выполнения кода, они будут помещены в окружение. В логах будет отображен список созданных объектов.
- Если объекты не были созданы, лог будет содержать сообщение о том, что окружение чисто.

Ctrl + R

Очистка окружения.

Полностью очищает окружение от ранее созданных объектов.

F1

Помощь с API текущего DCC.

Открывает главный веб-браузер и отображает главную страницу с текущим API DCC.

Примечание: Редактор является гибридом между IDE и REPL моделями и может использоваться для детального анализа созданных объектов.

Горячие клавиши редактирования текста

Tab

Эквивалент 4 пробелам.

Поддерживает многострочный режим.

Shift + Tab

Обратное действие.

Определяет количество пробелов перед текстом. Удаляет отступы по 4 пробела.

Поддерживает многострочный режим.

Ctrl + D

Дублировать строку.

Нажатие дублирует текущую строку и помещает её ниже курсора, перемещая курсор на новую строку.

Поддерживает многострочный режим.

Ctrl + /

Закомментировать строку (убрать комментарий).

Создает комментарий (#) из строки, где находится курсор. Нажатие снова убирает комментарий.

Поддерживает многострочный режим.

Ctrl + +
Ctrl + -

Изменить размер шрифта.

(Ctrl + +) увеличивает размер шрифта.
(Ctrl + -) уменьшает размер шрифта.

Ctrl + ↑
Ctrl + ↓

Переместить курсор к навигационной закладке вверх/вниз.

(Ctrl + ↑) перемещает курсор к навигационной закладке вверх.
(Ctrl + ↓) перемещает курсор к навигационной закладке вниз.
