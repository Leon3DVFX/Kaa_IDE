# Kaa_IDE
A Qt-based (PySide6) IDE for Python scripting in Houdini, Maya, and 3ds Max.

## Features
- Python editor with DCC integration
- Qt (PySide6) based UI
- Designed for technical artists and developers

## Tech stack
- Python 3
- PySide6 (Qt)
- Houdini / Maya / 3ds Max APIs

## DCCs
 - Houdini 21+
 - Maya 2025+
 - 3dsMax 2025+
