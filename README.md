# Kaa_IDE
A Qt-based (PySide6) IDE for Python scripting in Houdini, Maya, and 3ds Max.

## Features
- Python editor with DCC integration
- Qt (PySide6) based UI
- Designed for technical artists and developers

## Tech stack
- Python 3.x
- PySide6 (Qt)
- Houdini / Maya / 3ds Max APIs

## Supported DCCs
 - Houdini 21+
 - Maya 2025+
 - 3dsMax 2025+

## Instalation
1. **Before installation**, download and install the main font [JetBrains Mono](https://www.jetbrains.com/lp/mono/).
2. Check out the video ["Installation Procedure"](https://youtu.be/msa-EqxcmOQ).
3. Using the video, create short scripts to run in your editors.
